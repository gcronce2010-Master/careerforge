# CareerForge AI

![CareerForge AI Banner](https://picsum.photos/seed/99/1200/400)

CareerForge AI is a professional portfolio builder designed to help tech enthusiasts and professionals craft compelling career narratives using Generative AI.

## Features

- **AI-Powered Bio Generation**: Instantly create a professional "About Me" section tailored to your goals.
- **Experience Refinement**: Use AI to polish your work history and project descriptions, highlighting transferable skills.
- **Real-time Preview**: See your portfolio update live as you type.
- **Cloud Sync**: Automatic saving to Firebase Firestore ensures your progress is never lost.
- **Export Options**: Download your portfolio as Markdown or Plain Text for easy sharing.

## Tech Stack

- **Next.js**: App Router for high-performance React applications.
- **Genkit**: Google's AI framework for managing LLM prompts and flows.
- **Firebase**: Firestore for real-time data persistence.
- **ShadCN UI**: Modern, accessible UI components.
- **Tailwind CSS**: Utility-first styling for a clean, responsive design.

## Getting Started

To get started, simply run the development server:

```bash
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) in your browser to start building your portfolio.
