# **App Name**: CareerForge AI

## Core Features:

- Structured Content Input Forms: Provides intuitive, guided forms to collect user's personal details, professional experience, education, skills, and project information.
- AI-Powered About Me Tool: Utilizes an AI tool to generate a professional, confident, and compelling 'About Me' section based on user-provided background, interests, and career goals.
- AI-Driven Experience Refinement Tool: An AI tool that enhances and rephrases raw work history and project descriptions to highlight transferable skills, key achievements, and professional impact.
- Dynamic Skill & Education Formatting: Automatically organizes and formats skills, educational background, and certifications into clear, scannable sections following professional portfolio standards.
- Portfolio Content Preview: Offers a real-time, responsive preview of the generated portfolio content, allowing users to see their polished profile instantly.
- Exportable Content Output: Enables users to download the complete, polished portfolio text content in web-ready formats (e.g., Markdown or plain text) for easy integration into a personal website.

## Style Guidelines:

- Light/Dark Scheme: A dark theme is chosen to reflect a modern and tech-focused aesthetic. The visual design conveys professionalism and a bold, creative edge.
- Primary color: #618CE6. This is a vibrant, deep blue that conveys professionalism, depth, and clarity, designed to stand out on a dark background.
- Background color: #2A2D35. A dark, desaturated blue-grey hue serving as a modern and immersive backdrop that complements the primary color.
- Accent color: #9C70F3. A bright, electric violet hue chosen to add a bold and creative spark, drawing attention to key interactive elements and highlights without being a saturated green or teal.
- Headline and Body Font: 'Space Grotesk' (sans-serif) for its computerized, techy, and modern aesthetic, ensuring content is professional, clear, and scannable.
- Use minimalistic, line-art style icons that are clean and modern, consistent with a technology-focused and professional brand.
- A sleek single-page scroll layout that efficiently guides users through the portfolio content. Key sections like education and projects will utilize a card-based design within a responsive grid for visual organization.
- Incorporate subtle, smooth transitions and micro-interactions for section changes and interactive elements, providing a polished and engaging user experience.
